# _Zenith_

Zenith is an AI-powered Event/Task Manager for the web. The concept is simple; you put in a prompt and natural language processing of AI will transform your prompt into an event!

# Technologies used

- SvelteKit
- TailwindCSS + daisyui
- Vercel AI SDK
- Drizzle ORM
- Neon DB
- Resend
- SvelteKit-Superforms
- zod
- Lucia
